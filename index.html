come iniziare ad usare sass con terminale macos!

Primi passi per creare una repository tramite GitKraken, aggiungendo poi LaravelMix per usare SASS:

  • 1 - Creare repository tramite INIT
  • 2 - Aprire il terminale, cercare la cartella creata tramite GitKraken e digitare 'npm init -y' per inizializzare il progetto NPM
  • 3 - Senza cambiare directory, digitare 'npm install laravel-mix --save-dev', così da installare il pacchetto LaravelMix
  • 4 - Ancora, senza cambiare directory, su terminale digitare 'touch webpack.mix.js'
  • 5 - Aprire adesso il proprio IDE ed aprire la cartella creata con GitKraken
  • 6 - Aprire il file 'webpack.mix.js' e scrivere all'interno 'let mix=require('laravel-mix')' e 'mix.sass('src/app.scss', 'dist/')' così da definire dove andare a prendere i file SCSS per trasformarli in CSS
  • 7 - A questo punto, creare le cartelle 'src' e 'dist' dove andranno all'interno rispettivamente il file SCSS ed il file CSS
  • 8 - Adesso, digitare su terminale 'npx mix', cioè il comando che lancia LaravelMix, trasformando così i file SCSS in file CSS
  • 9 - Infine, subito dopo digitare di nuovo (sempre su terminale) 'npx mix', successivamente digitare 'npx mix watch' (così da evitare di ripetere il comando 'npx mix' ad ogni modifica SCSS che vogliamo 'controllare' su web)
